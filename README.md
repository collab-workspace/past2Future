Past2Future is a web-based platform that connects historical insights with future-oriented perspectives through global events and traveler experiences.

The application enables users to explore past and future events across different countries, interact with an interactive globe, and share thoughts categorized as historical reflections or future predictions.

---

##  Features

- Interactive globe with country-based event visualization  
- Past and future events listing  
- Traveler‚Äôs Thoughts (History / Prediction based comments)  
- Authentication and authorization (Admin / User roles)  
- Admin panel for managing events, countries, and content  
- Dark theme user interface  

---

## Tech Stack

- **Backend:** ASP.NET Core MVC  
- **Frontend:** Razor Views, JavaScript, Bootstrap  
- **Database:** SQL Server with Entity Framework Core  
- **Authentication:** ASP.NET Identity  
- **Version Control:** Git & GitHub

##  Setup & Installation

### 1. Clone the repository
```bash
git clone https://github.com/collab-workspace/past2future.git
````

### 2. Create your own configuration file

Copy `appsettings.example.json` and rename it to `appsettings.json`.

### 3. Update the database connection string

Edit the connection string inside `appsettings.json`.

### 4. Apply database migrations

```bash
Update-Database
```

### 5. Run the project

```bash
dotnet run
```
